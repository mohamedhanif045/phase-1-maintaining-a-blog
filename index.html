Basic Math Functions in JavaScript

Learn how to implement basic arithmetic operations using JavaScript.

Addition

function add(a, b) {
    return a + b;
}

Subtraction

function subtract(a, b) {
    return a - b;
}

Multiplication

function multiply(a, b) {
    return a * b;
}

Division

function divide(a, b) {
    return a / b;
}
